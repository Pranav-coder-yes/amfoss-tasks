# Expense Tracker Documentation
## Overview
The Expense Tracker application is a tool designed to help users manage their finances by tracking income and expenses. It provides features like adding, viewing, editing, and deleting transactions, categorizing expenses, and generating reports.

# Features
Add Transactions: Record income and expenses with details like date, category, and amount.
View Transactions: Display a list of all transactions with sorting and filtering options.
Edit/Delete Transactions: Modify or remove existing transactions.
Categorization: Organize transactions by categories like food, utilities, etc.
Reports: Generate monthly or yearly expense reports.


# Installation
Clone the Repository: git clone https://github.com/gauravgunjal71/Expense-Tracker.git
Install Dependencies: pip install -r requirements.txt
Run Migrations: python manage.py migrate
Start the Server: python manage.py runserver


# Usage
Adding Transactions: Navigate to the "Add Transaction" section to record a new income or expense.
Viewing Transactions: Use the "Transactions" page to view all records.
Editing Transactions: Click on a transaction to edit details.
Generating Reports: Access the "Reports" section to view spending summaries.


# Directory Structure
/migrations: Contains database migration files.
/templates: Holds HTML templates for the application's front end.
/static: Contains static files like CSS and JavaScript.
/views: Defines the logic for different app routes and pages.


# Technologies Used
Backend: Django
Frontend: HTML, CSS, JavaScript
Database: SQLite (default)
